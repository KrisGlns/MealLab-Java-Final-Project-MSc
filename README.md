# MealLab - Java Recipe Management System

MSc Java Course Final Project - University Assignment

## Project Description

MealLab is an intelligent kitchen management application that helps users:
- Search for recipes using TheMealDB API
- Get random recipe suggestions
- Manage favorite and cooked recipe lists
- View detailed cooking instructions with ingredients

## Project Structure

This project consists of two main components:

### Part A: MealLab API (`meallab-api/`)
A Maven library that provides:
- REST API client for TheMealDB
- POJO classes for recipe data
- JSON deserialization
- Exception handling
- JUnit tests

### Part B:  MealLab App (`meallab-app/`)
A JavaFX desktop application that provides:
- User-friendly GUI for recipe search
- Random recipe suggestions
- Favorite and Cooked lists management
- Persistent data storage (JSON)
- Detailed recipe view with instructions

## Technologies Used

- **Java 25**
- **Maven** - Dependency management and build tool
- **JavaFX** - GUI framework
- **TheMealDB API** - Recipe database
- **JUnit 5** - Unit testing
- **Jackson** - JSON processing
- **Git/GitHub** - Version control

## Project Requirements

- JDK 11 or higher (using JDK 25)
- Maven 3.6+
- IntelliJ IDEA (recommended)
