# Project Documentation

This folder contains all project documentation and reports.

## Contents

- Project report (DOCX and PDF formats)
- API exploration notes
- Architecture diagrams
- Screenshots of the application
- JUnit test results
- User guide
- Development notes

## Report Structure

The final report will include:

1. Description of REST API usage and POJO implementation
2. Application implementation approach analysis
3. Feature-by-feature implementation details
4. Application screenshots and demonstration
5. JUnit test results and examples
6. Tools and technologies used (Postman, IntelliJ, Maven, Git)
7. GitHub repository link
8. Team member contributions (if applicable)
